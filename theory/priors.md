Parameter Priors and Admissible Ranges

Cosmological parameters:
H0 > 0

Density parameters:
Omega_m0 >= 0
Omega_r0 >= 0
Omega_Lambda0 >= 0
Omega_Phi0 = 1 - (Omega_m0 + Omega_r0 + Omega_Lambda0)

Deformation parameters:
alpha in [-1, 1]
beta in [-1, 1]

Stability conditions:
H(z) finite for all z >= 0
|Phi(z)| / H(z)^2 bounded for all z >= 0

Identifiability:
alpha and beta are degenerate with Omega_Phi0 at z = 0
Degeneracy broken by redshift evolution

Priors:
Flat priors on alpha and beta within admissible ranges
Standard observational priors on H0 and density parameters
