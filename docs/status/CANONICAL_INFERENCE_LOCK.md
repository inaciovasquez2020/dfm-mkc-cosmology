# Canonical Inference Lock

Status: Conditional

This repository implements cosmological model comparison and parameter inference.

Constraints:

1. All inference results are conditional on the chosen likelihood and priors.
2. Parameter degeneracies are expected and must be explicitly reported.
3. No claim of unique cosmological reconstruction is allowed without identifiability guarantees.
4. Model comparison outputs remain statistical, not deterministic truth claims.

Do not promote beyond Conditional on repository-surface evidence alone.
